# 七、最终建议

你现在应该：

# 以“大屏数据项目”为核心进行整理。

建议最终Excel结构：

| 大屏名称 | 模块名称 | 数据项目 | 指标口径 | 刷新频率 | 展示形式 |
| ---- | ---- | ---- | ---- | ---- | ---- |

例如：

| 大屏名称    | 模块名称  | 数据项目   |
| ------- | ----- | ------ |
| 零售经营驾驶舱 | 顶部KPI | AUM总资产 |
| 零售经营驾驶舱 | 存款经营  | 活期余额   |
| 财富经营大屏  | 财富产品  | 基金销量   |
| 风险监控大屏  | 风险预警  | 高频交易   |

这样后续：

* 大屏设计
* 数据开发
* ADS建模
* Echarts开发
* 百度文心生成
* DataV配置

都会非常容易。
